Before you begin

This is an iOS class. To complete the coding exercises, you will need a Xamarin.iOS development environment with either the iOS Simulator or an iOS device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

In iOS, it's fairly rare to come across applications (that aren't games) which do not use a Table View. This commonly used control lets an application display a scrollable list of data which can be manipulated and often forms the basis of master/detail views. This class will go through the basics of using Table Views, getting them into your UI, populating them with data and performing some basic customizations.

Objectives

  1. Work with built-in editing operations
  2. Add support for modern editing operations
  3. Integrate a search bar

Lecture

IOS215 - Editing TableViews.

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Support editing operations on a Table View
    Exercise
    In this first exercise, you will modify an existing application to turn on the built-in editing features supported by a Table View.
  2. Committing editing operations
    Exercise
    Next, you will finish the Table View editing support by implementing the CommitEditingStyle method and refresh the Table View data.
  3. Add support for swipe-gesture edit actions
    Exercise
    A relatively new feature in iOS adds support for custom "swipe" actions. In this exercise, you will update the Expense Tracker application to support custom edit actions.
  4. Add support for the "pull-to-refresh" gesture
    Exercise
    In this group exercise, we will use the pull-to-refresh gesture, which is a popular addition to Table Views that display external data and get new data from the external source.
  5. Add search support to a Table View Controller
    Exercise
    In this final exercise, you will add a search bar to a Table View using UISearchController.
  6. Scope your searches with a scope bar
    Exercise
    In this optional homework assignment, you will integrate some category filters in the search experience by including a Scope Bar.
